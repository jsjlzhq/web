1、C/S 与 B/S
	C/S : c client客户端     s  server 服务器  客户端服务器架构
	
	B/S : b  browser  浏览器   s  server 服务器   浏览器服务器架构
	对比：  
		cs: 1、需要下载安装   2、客户端和服务器端都需要更新   3、开发和维护的成本高
		bs: 1、不需要下载安装   2、只需要更新服务器端   3、开发和维护的成本低

4、html	
		Hypertext Markup Language: 超文本标签语言或者超文本标记语言
		
		作用：不需要编译，直接由浏览器执行
		文件后缀：.html   或者  .htm  （推荐使用.html）
		修改文件类型：
			组织===》文件夹和搜索选项====》查看===》隐藏已知文件的类型的扩展名（把前面的对勾去掉就可以了）

6、标签：
		以<>包起来的特殊字符串，通常情况下有开始标签和结束标签
			分类：
				双边标签：<标签名>内容</标签名>
				单边标签：<标签名 />   (标准格式)
		
   属性：对标签进行一定的修饰，让其功能更加完善。
			<标签名 属性名1="值1" 属性名2='值2' 属性名3=值3>内容</标签名>
				推荐大家属性的值用双引号包裹起来。
   
   元素：一个完整的标签就是一个元素。
特性：容错性强   已预定义  全部小写   英文半角状态   标签直接可以嵌套

7、指定字符集
	 <meta charset="utf-8" />
	 解决乱码问题的步骤：
		1、将文件的格式保存为utf-8 
		2、浏览器进行编译的时候也以utf-8的格式进行编译
			第一种方案： 在文件的开始位置写上<meta charset="utf-8" />
			第二种方案：在浏览器里面来修改一下编码格式
	
8、全局架构标签
	<html>
		<head>
			<meta charset="utf-8" />
			<title>网易</title>
		</head>
		<body></body>
	</html>
	【注】：
			1、所有的内容必须写到html标签中
			2、head标签中的内容不会显示到浏览器中
			3、body标签中的内容会显示到浏览其中
	html注释：方便自己日后查看自己的代码或者方便团队开发时，同事查看。
		注释不会显示到浏览器中。
	body标签常用的属性：
		bgcolor:背景色
		background：背景图片
		【注】：当同时设置了背景图片和背景颜色之后，背景图片会优先显示（默认以平铺的方式显示到浏览器中）
		text：文本内容的颜色。
		leftmargin:文本内容的左边距
		topmargin：文本内容的顶部边距
		link 、alink、 vlink这三个属性需要配合a标签来使用。
		link：文本内容默认的颜色
		alink：鼠标按下去文本内容的颜色
		vlink：访问过后，文本内容的颜色
		
	class id  name  style    html文件中任何一个标签都具备的属性，主要是配合后面的css和js来使用。

10、常用标签之文本修饰
	标题：h1 -- h6  <h1></h1>----<h6></h6>   <h1></h1>最大  <h6></h6>最小    属性：align  控制水平方向的位置
	加粗：<b></b><strong></strong>
	斜体：<i></i><cite></cite><em></em>
	下划线标签<u></u>    
	删除线标签<s></s>  
	上标：<sup></sup>  
	下标：<sub></sub>   
	滚动标签：<marquee></marquee>
		scrollamount：滚动速度
		direction：滚动方向
		loop：循环次数
		width：宽度
		height：高度
		bgcolor：背景颜色
		
	字体：<font></font>   
		size:字体大小   默认值从1到7，
		face：字体的样式
		color：字体的颜色。
	颜色的两种表示方式：（color）
		 第一种：英文单词
		 第二种：RGB   三原色   红绿蓝     通过16进制实现  详见手册调色原理.
11、常用标签之格式控制
	段落标签<p></p>   会独立占一行，默认在标签的前面和后面加了一个<br />换行标签。
	
	水平线：<hr /> 
			width: 宽度（值可以用绝对值也可以用相对值，不需要写单位）
			size:水平线的厚度
			color:水平线的颜色
			align:水平方向的位置  （left \ right \ center(默认值)）
	无序列表：<ul><li></li></ul>  (type:square\disc\circle)
		type:来控制li标签前面图形显示类型
		square:正方形
		circle：空心圆
		disc：实心圆  （默认值）
	有序列表：<ol><li></li></ol>  (type(类型，默认前面的是数字):a\A\I\i  
		start：从第几个开始，后面的值必须是数字)  
	自定义列表：<dl><dt></dt><dd></dd></dl>
	
12、无意义标签
	块标签：<div></div>   <header></header> <section></section> <footer></footer> （独立占一行，可以设置他的宽和高，默认不写宽度，占整行）
	行内标签：<span></span>   （默认会在一行里面显示，不能设置宽和高，需要靠里面的内容把标签撑起来）

13、url（统一资源定位符）  （重点）
	例子：http://www.jd.com:80/index.html?name="ersha"&sex="yaoren"
	组成部分：协议名://域名:端口号/文件名？参数名=值1&参数名2=值2
		协议：  http://   https://   ftp://    file:///
		域名（主机名）：网站的名字
		端口号：取值范围：0---65535，  通常使用的是0-1024(已经全部被占用),
			http://    80    https:// 443   ftp://   20
		文件名：请求文件的名字
		参数：通常情况下以键值对的形式出现   多个参数之间使用&连接。
